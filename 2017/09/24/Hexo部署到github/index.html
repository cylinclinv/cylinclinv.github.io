以上发布完成后,在浏览器输入cylinclinv.github.io就打开了自己的博客

域名配置

cmd输入ping cylinclinv.github.io,获得github的公网ip地址

在你购买域名的地方配置这个公网ip解析,这个ip地址是指向github网站而不是你博客

想根据这个找到你博客地址,就需要在github pages配置自定义域名(custom domain)

最后在blog目录/source下创建CNAME文件(没有后缀),内容输入你的域名,这一步很重要,不然每次通过 hexo d 部署,都需要重新配置custom domain
